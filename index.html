JavaScript Functions

Learn everything you need to know about JavaScript functions and Date methods.

What is a Function?

A JavaScript function is a block of code designed to perform a particular task. Functions are reusable and allow you to structure your code in a modular way.

// Basic function syntax
function greet() {
  console.log("Hello, World!");
}

Types of JavaScript Functions

1. Named Function

Named functions are declared with a specific name and can be called by that name anywhere in your code.

function sayHello() {
  console.log("Hello!");
}

2. Anonymous Function

Anonymous functions do not have a name and are often used as arguments or assigned to variables.

const greet = function() {
  console.log("Hello!");
};

3. Arrow Function

Arrow functions provide a shorter syntax and don't bind their own `this` context.

const greet = () => {
  console.log("Hello!");
};

Working with Date Methods

The Date object in JavaScript provides methods to work with dates and times. Here are some common Date methods: 

// Current date and time
const now = new Date();
console.log(now);

// Get year, month, day
console.log(now.getFullYear());
console.log(now.getMonth());
console.log(now.getDate());

Real-World Examples

Here's an example using a function with Date methods to display the current time:

function displayTime() {
  const now = new Date();
  return now.toLocaleTimeString();
}
